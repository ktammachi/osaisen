# Foreigners Online Shrine

This repository can be deployed to **GitHub Pages** automatically via Actions.

## What was added
- `.github/workflows/deploy-pages.yml` deploys `online-shrine-for-foreigners/` to GitHub Pages.

## How to enable (one-time)
1. Push this branch to GitHub.
2. Open **Settings → Pages**.
3. Set **Source** to **GitHub Actions**.
4. Run or re-run the **Deploy Online Shrine to GitHub Pages** workflow.

## Published URL
After deployment, the app is available at:

- `https://<your-github-username>.github.io/<repo-name>/`

(For project pages, the path maps to `online-shrine-for-foreigners/index.html` as site root because the workflow uploads that folder directly.)
