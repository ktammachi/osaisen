# Foreigners Online Shrine

This repository is deployable to **GitHub Pages** via GitHub Actions.

## Deploy target
- The Pages workflow deploys the repository root (`.`).
- Root URL (`/`) redirects to `online-shrine-for-foreigners/index.html`.

## One-time setup
1. Merge this branch into `main`.
2. In GitHub, open **Settings → Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to `main` (or run the workflow manually).

## URLs after deploy
- Main entry: `https://<your-github-username>.github.io/<repo-name>/`
- Direct page: `https://<your-github-username>.github.io/<repo-name>/online-shrine-for-foreigners/index.html`
